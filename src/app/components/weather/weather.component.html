Weather forecast

This is a simple weather forecast application which has only desktop layout jet.

In this project I use two different API to get the required data. Google places API provides the coordinates for the Open Weather API to get the weather data. This application receives 40 weather log, one for every 3 hours in the next 120 hours, This feature shows the weather for the next 5 day.

Firstly I use Google places API to implement an autocomplete component mechanism which makes easier to find a city and in the same time all options given by the API are valid. When a place name is chosen the API gives back photos from that particular city that I use as backgrounds for the slider and I can collect the latitude and longitude of the city.

I use these data to make a request to the Open Weather API. Two interfaces and three costume pipes help to handle the data.

This application contains three components and two services in order to achieve reusability and encapsulate the logic. This project has simple error handling options in case of potential problems.

How to use weather forecast app

  • It starts with an image which greats the user
  • Type a city name than click on an option given by the autocomplete feature.
  • The application shows a slider that contains the weather forecast data

Időjárás előrejelzés

Ez egy egyszerű időjárás előre jelző applikáció, ami jelenleg még nem rendelkezik responsive elrendezéssel.

Két külön API segítségével működtetem a projektet melyek a Google places API, ami a helyadatokat biztosítja az Open Weather API számára az időjárás adatokhoz. Az alkalmazás 40 időjárás bejegyzést kap 3 óránként egy adat tömböt a következő 120 órára. Igy rendelkezik a következő 5 nap adataival.

Elsőként a Google places API megvalósít egy automatikus kiegészítés funkciót, mely megkönnyíti a városok megtalálását valamint egyúttal validálja is a helyszín létezését. Miután a legördülő menünől ki lett választva a kívánt helyiség az API visszaadja annak geolokációját továbbá képeket biztosít a dizájnhoz, amiket háttérnek használok.

Ezen adatokkal le tudom kérni az előre jelzést az Open Weather API-n keresztül. Két interface és három pipe segít az adatok kezelésében.

Az applikációt három komponens és két service alkotja. Ezek újrahasználási célokat és az üzleti logika elzárását valósítják meg. A projekt továbbá egy egyszerű hibakezelést is végez az esetleges problémák megoldása végedt.

Az applikáció használata

  • Az oldal egy üdvözlőképpel fogad
  • Itt csak egy kívánt helyiség nevet kell az input mezőbe beírni majd kiválasztani a legördülő menüből a kívánt várost
  • Ezután az oldal a slider segítségével jeleníti meg a következő 5 nap időjárását.
