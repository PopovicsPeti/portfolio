Space travel website

This project was built by the instructions of Kevin Powell, Build a Space Travel Website course on Scrimba. The design of the web page is completely his idea. This project is a vanilla JS and mostly CSS project on the Scrimba education site.

In my case Angular framework was used to create the project. It has changed several solutions in the codeing methods. The Crew page was only mentioned but the lessons did not have complex details to build that page and the Destination page was completely Homework.

This project focuses mainly to the design. It has some HTTP requests from the database because the sliders get the information from a database and Angular methods like ngFor handle the data. This site has no user features it is only a landing page of a fictive “commercial space travel agency”.

Űr ügynökség

Ezt a projektet Kevin Powel munkássága nyomán készítettem el, a Scrimba Frontend oktató weboldal "Build a Space Travel Website" című kurzusának segítségével. A teljes weboldal dizájnja az ő ötlete és útmutatásai alapján készült. Ez a weboldal szimpla Java Script és CSS oktatatására és gyakorlására szolgál.

Az én esetemben Angular keretrendszer használatával történt a megvalósítás, mely alapjaiba változtatta meg a projekt felépítését és a logika használataát. Az oktatóanyagban a "crew" oldal kódolása már csak nagy vonalakban van említve míg a "destination" oldal önálló feladatként van feladva.

Ez egy dizájn fókuszú weboldal, egy fiktív űrügynökség reklám oldala. A projekt nem tartalmaz regisztrációhoz és felhasználókhoz köthető tartalmakat. Azonban az adatok lekérése egyszerű HTTP kérésekkel van megoldva Firebase-ről az eredeti anyag JS-be való beleégetése helyett. A lekért adatok menedzselése az angular adta lehetőségek igénybevételével történikaz oldalon.
